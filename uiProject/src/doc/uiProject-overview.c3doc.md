Title: Welcome to C3 AI Academy Forecasting Project Solution

Welcome to the C3 AI Academy Forecasting Project Solution! This application is purely for internal training team reference.

## Setup steps
1. Deploy artifact to your environment.
2. Start Jupyter after the artifact is deployed.
3. Find the notebooks in `/academy`.


## Notebook overview
**setup.ipynb**: The initial setup notebook that prepares the data for the project. Includes data generation, ingestion, and `Feature.Set` materialization. **Run this notebook first** before proceeding to any others.

**project.ipynb**: The main notebook containing the core activities of the project.

**project-solution.ipynb**: A complete version of `project.ipynb` that includes solutions to all activities. This notebook is for training team reference.

**introduction-to-the-tabular-feature-creator.ipynb**: A concise tutorial introducing the `TabularFeatureCreator` and `TabularFeatureOperation` classes from the Time Series Analysis toolkit. This notebook supports the "Feature Engineering" section of the main project by demonstrating how to generate and manipulate tabular features effectively.
